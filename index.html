Guide pour utilisé GitHub et Pages

à l'attention des étudiant OCR

Pour bien demmarer

-1- Créer votre premier repository

  1. Rendez-vous sur l'onglet "Répositories"
  2. Cliquez sur "New"
  3. Choisissez un nom adapté à votre projet
  4. Validez

-2- Transformez votre repo en gh-pages

  1. Rendez-vous sur votre repo puis dans "Settings"
  2. Ajoutez ".github.io" au nom du repo et clickez sur "Rename"
  3. Clicker sur l'onglet "Pages" et dans le cadre vert vous retrouvez le lien de votre site web. En cliquant dessous verfiier que votre site fonctionne (page blanche si le repo est vide)

-3- Récuperer voter depot pour travailler en local

  1. Ouvrez-votre editeur
  2. Ouvrez un terminal
  3. utilisé la commande "git init"
  4. puis git clone "lien du depot"

Envoyez son travail sur GitHub

Premier commit

  1. Creez votre fichier index.html
  2. Utilisez la commande "git add nom_du_fichier"
  3. Créer le commit associer avec "git commit -m "action réaliser"
  4. Rendez-vous sur la branch principal (ou la branche souhaitez) avec "git branch -M nom_de_la_branch"
  5. puis "git remote add origin https://github.com/username/nom_du_depot.git"
  6. puis "git push -u origin nom_de_la_branch"
  7. Si tous c'est bien passé le fichier index.html est sur votre depot
