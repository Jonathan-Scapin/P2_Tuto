Démmarer un projet GitHub

Créer votre premier repository
  1. Rendez-vous sur l'onglet "Répositories"
  2. Cliquez sur "New"
  3. Choisissez un nom adapté à votre projet
  4. Validez
Transformez votre repo en gh-pages
  1. Rendez-vous sur votre repo puis dans "Settings"
  2. Ajoutez ".github.io" au nom du repo et clickez sur "Rename"
  3. Clicker sur l'onglet "Pages" et dans le cadre vert vous retrouvez le lien de votre site web. En cliquant dessous verfiier que votre site fonctionne (page blanche si le repo est vide)
Récuperer voter depot pour travailler en local
  1. Ouvrez-votre editeur
  2. Ouvrez un terminal
  3. utilisé la commande

    code :

                                    git init
  4. puis

    code :

                                    git clone "url du depot"

Envoyez son travail sur GitHub

Premier commit
  1. Creez votre fichier index.html
  2. Utilisez la commande

    code :

                                        git add nom_du_fichier
  3. Créer le commit associer avec

    code :

                                        git commit -m "action réaliser"
  4. Rendez-vous sur la branch principal (ou la branche souhaitez) avec

    code :

                                        git branch -M nom_de_la_branch
  5. puis

    code :

                                        git remote add origin https://github.com/username/nom_du_depot.git
  6. puis

    code :

                                        git push -u origin nom_de_la_branch
  7. Si tous c'est bien passé le fichier index.html est sur votre depot
